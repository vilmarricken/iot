Comando byte(0-255)-Parâmetros


Comando: 1 - Retorna informações do dispositivo
Parâmetros:
Retorno:
 - byte - indicando a versão do firmware da placa
 - byte - indicando o tipo da placa 
   - 1 = ESP8266
   - 2 = ESP8266 NODEMCU
 - byte - para indicar se a porta esta em uso
   - Cada bit 1 indicará a porta que esta em uso
 - byte - para indicar o tipo da porta I/O ou sensor
   - Bit 1 para I/O
   - Bit 0 para sensor ou fora de uso
 - byte - para indicar o modo de cada porta configurada 
   - Cada bit 1 indicará a porta que esta liga (portas que são de leitura serão sempre 0)

Comando: 2 - Configurar portas
Parâmetros
 - byte - Porta que esta conectado o dispositivo
 - byte - Tipo do dispositivo
   - 0 - sensor
   - 1 - relê
